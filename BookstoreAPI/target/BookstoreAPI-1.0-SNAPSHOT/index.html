Bookstore API - Main Endpoints

Authors

POST /api/authors
Create a new author
GET /api/authors
Get all authors
GET /api/authors/{id}
Get author by ID
PUT /api/authors/{id}
Update author by ID
DELETE /api/authors/{id}
Delete author by ID
GET /api/authors/{id}/books
Get books by author ID

Books

POST /api/books
Create a new book
GET /api/books
Get all books
GET /api/books/{id}
Get book by ID
PUT /api/books/{id}
Update book by ID
DELETE /api/books/{id}
Delete book by ID

Customers

POST /api/customers
Create a new customer
GET /api/customers
Get all customers
GET /api/customers/{id}
Get customer by ID
PUT /api/customers/{id}
Update customer by ID
DELETE /api/customers/{id}
Delete customer by ID

Cart

POST /api/customers/{customerId}/cart/items
Add item to cart
GET /api/customers/{customerId}/cart
Get customer's cart
PUT /api/customers/{customerId}/cart/items/{bookId}
Update cart item quantity
DELETE /api/customers/{customerId}/cart/items/{bookId}
Remove item from cart

Orders

POST /api/customers/{customerId}/orders
Place an order
GET /api/customers/{customerId}/orders
Get all orders for a customer
GET /api/customers/{customerId}/orders/{orderId}
Get specific order by ID
